HTML

One of Angular's main jobs is to dynamically render html to the screen

Interpolation and Template Expressions

This code
{{"{{ 2 + 3 \}\}"}}
will result in "{{ 2 + 3 }}" written to the screen in place of itself.

You can also display variables from the controller

The code
{{"{{ myVariable \}\}"}}
Will result in displaying a primitive as a string on the template.
Note* if you want to display a complex type you have to stringify it first or add the json pipe
{{"{{ myVariable | json \}\}"}}
